# EcoNudge AI

Hyper-local pollution nudges for cleaner campuses & cities.

This scaffold contains a Next.js frontend, a Node.js backend prepared for Firebase/Supabase integration, and a small AI engine with Python scripts for pollution analysis and task recommendations.

Structure:
- frontend/: Next.js app (app router)
- backend/: Node.js API and Firebase config
- ai-engine/: Python scripts and dependencies
- docs/: architecture and slides placeholder
